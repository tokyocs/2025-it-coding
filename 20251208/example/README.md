# サンプルファイル - テーマ切り替えアプリ

このフォルダには、第5回の授業で使うテーマ切り替えアプリのサンプルファイルが入っています。

## 📁 フォルダ構成

```
example/
├── basic/           ← テーマ切り替え（シンプル版）
│   ├── index.html
│   ├── style.css
│   └── script.js
├── full/            ← テーマ切り替え（フル版）
│   ├── index.html
│   ├── style.css
│   └── script.js
└── omikuji/         ← おみくじアプリ
    ├── index.html
    ├── style.css
    └── script.js
```

## 🎯 サンプルの種類

### 1. basic/ - シンプル版（まずはここから！）

**ボタン：**
- 「次のテーマ」ボタンのみ

**テーマ：**
- 海、森、夕焼けの3種類

**こんな人向け：**
- 最初に試す人
- シンプルなコードを読みたい人
- 自分でボタンを追加してみたい人

**開き方：**
1. `basic` フォルダを開く
2. `index.html` をダブルクリック

---

### 2. full/ - フル版（ボタン全部入り）

**ボタン：**
- 「◀ 前」ボタン
- 「🎲 ランダム」ボタン
- 「次 ▶」ボタン

**テーマ：**
- 海、森、夕焼け、夜空、さくらの5種類

**こんな人向け：**
- 完成版を見たい人
- ランダム機能を試したい人
- コードを参考にしたい人

**開き方：**
1. `full` フォルダを開く
2. `index.html` をダブルクリック

---

### 3. omikuji/ - おみくじアプリ 🎋

**機能：**
- ボタンを押すとランダムにおみくじが引ける
- 結果によって文字の色が変わる
- メッセージも表示される

**いじれるところ：**
- 結果のテキスト（「大吉」→「超ラッキー」など）
- メッセージ（好きな言葉に変えられる）
- 背景色・文字色

**こんな人向け：**
- ランダムの仕組みを理解したい人
- 自分だけのおみくじを作りたい人
- テーマ切り替えの応用を見たい人

**開き方：**
1. `omikuji` フォルダを開く
2. `index.html` をダブルクリック
3. 「おみくじを引く」ボタンを押してみよう！

## 🌟 初めての人向け：まず動かしてみよう！

### ステップ1：シンプル版を開く

1. `example` → `basic` フォルダを開く
2. `index.html` をダブルクリック
3. ブラウザでアプリが開きます！

### ステップ2：ボタンを押してみよう

1. **「次のテーマ」** ボタンを押してみよう
2. 色が変わったら成功！🎉
3. 何回か押して、色の変化を楽しもう

### ステップ3：フル版も試してみよう

1. `example` → `full` フォルダを開く
2. `index.html` をダブルクリック
3. **「🎲 ランダム」** ボタンを押してみよう！
4. ランダムに色が変わる！✨

### ステップ4：おみくじも試してみよう

1. `example` → `omikuji` フォルダを開く
2. `index.html` をダブルクリック
3. **「おみくじを引く」** ボタンを押してみよう！
4. 何が出るかな？🎋

### ステップ5：色を変えてみよう

1. `basic` フォルダの `script.js` を右クリック → 「メモ帳」で開く
2. 上の方にある色コード（`#0077be` など）を探す
3. 好きな色コードに変えてみる
4. 保存する（Ctrl+S）
5. ブラウザで更新する（F5）

色が変わったら成功！✨

## 🎨 いじれるところ

`script.js` を開くと、最初の方にこんなコードがあります：

```javascript
const themes = [
    {
        name: "海",              // ← ここを変える！
        background: "#0077be",  // ← ここを変える！
        text: "#ffffff"         // ← ここを変える！
    },
    // ...
];
```

### 変えられるもの

| 項目 | 説明 | 例 |
|------|------|-----|
| `name` | テーマの名前 | "海" → "空" |
| `background` | 背景の色 | "#0077be" → "#ff0000" |
| `text` | 文字の色 | "#ffffff" → "#000000" |

## 💡 使い方のコツ

### 色の組み合わせ

**暗い背景** → 白い文字（`#ffffff`）
**明るい背景**（黄色、黄緑など） → 黒い文字（`#000000`）

### テーマを追加するとき

```javascript
const themes = [
    { name: "海", background: "#0077be", text: "#ffffff" },
    { name: "森", background: "#228b22", text: "#ffffff" },
    { name: "夕焼け", background: "#ff6b6b", text: "#ffffff" },
    // ↑ ここにカンマを追加！
    { name: "新テーマ", background: "#色コード", text: "#ffffff" }
    // ↑ 最後にはカンマをつけない！
];
```

## 🔧 よくあるエラー

### 色が変わらない

1. ファイルを保存しましたか？（Ctrl+S）
2. ブラウザを更新しましたか？（F5）
3. 色コードは `#` から始まっていますか？
4. `"` で囲まれていますか？

```javascript
// ✅ 正しい
background: "#0077be"

// ❌ 間違い（#がない）
background: "0077be"

// ❌ 間違い（"がない）
background: #0077be
```

### ページが真っ白になった

JavaScriptにエラーがあるかもしれません。

1. ブラウザで F12 キーを押す
2. 「Console」タブを開く
3. 赤いエラーメッセージを確認
4. よくあるミス：
   - カンマ `,` を忘れた
   - 括弧 `{ }` の数が合っていない

### テーマが増えない

最後以外のテーマの後ろにカンマ `,` がありますか？

```javascript
// ✅ 正しい
{ name: "夕焼け", background: "#ff6b6b", text: "#ffffff" },
{ name: "宇宙", background: "#1a1a2e", text: "#ffffff" }

// ❌ 間違い（夕焼けの後ろにカンマがない）
{ name: "夕焼け", background: "#ff6b6b", text: "#ffffff" }
{ name: "宇宙", background: "#1a1a2e", text: "#ffffff" }
```
