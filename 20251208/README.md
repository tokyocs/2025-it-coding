## パート1：配列（はいれつ）ってなに？

### 配列は「データの棚」

前回のおみくじで少し出てきた**配列**を、もっと詳しく学びましょう！

```javascript
// これが配列！ [ ] の中にデータを入れる
const colors = ["赤", "青", "緑"];
```

| 番号 | 中身 |
|------|------|
| 0番目 | "赤" |
| 1番目 | "青" |
| 2番目 | "緑" |

**ポイント：番号は0から始まる！**

### 配列からデータを取り出す

```javascript
const colors = ["赤", "青", "緑"];

console.log(colors[0]);  // "赤" が表示される
console.log(colors[1]);  // "青" が表示される
console.log(colors[2]);  // "緑" が表示される
```

## パート2：テーマ切り替えの仕組み

### テーマのデータを配列で作る

```javascript
// テーマの配列
const themes = [
    {
        name: "海",
        background: "#0077be",
        text: "#ffffff"
    },
    {
        name: "森",
        background: "#228b22",
        text: "#ffffff"
    },
    {
        name: "夕焼け",
        background: "#ff6b6b",
        text: "#ffffff"
    }
];
```

**ポイント：**
- `name` → テーマの名前
- `background` → 背景の色
- `text` → 文字の色

### ボタンを押すと次のテーマに

```javascript
// 今のテーマ番号（0からスタート）
let currentTheme = 0;

// ボタンが押されたとき
button.addEventListener('click', function() {
    // 次のテーマに進む
    currentTheme = currentTheme + 1;
    
    // 最後まで行ったら最初に戻る
    if (currentTheme >= themes.length) {
        currentTheme = 0;
    }
    
    // テーマを適用する
    applyTheme();
});
```

### ★★★ チャレンジ5：「前のテーマ」ボタンを追加しよう

**やりたいこと：**
- 「前のテーマ」ボタンを追加して、戻れるようにしたい

**ステップ：**

1. **HTMLにボタンを追加（index.html）：**
```html
<button id="prevButton" class="btn">◀ 前のテーマ</button>
```

2. **JavaScriptに処理を追加（script.js）：**
```javascript
// 前のテーマボタンを取得
const prevButton = document.getElementById('prevButton');

// 前のテーマボタンが押されたとき
prevButton.addEventListener('click', function() {
    currentTheme = currentTheme - 1;
    
    // 最初より前に行ったら最後に戻る
    if (currentTheme < 0) {
        currentTheme = themes.length - 1;
    }
    
    applyTheme();
});
```

## 困ったときは

### 色が変わらない

1. ファイルを保存しましたか？（Ctrl+S）
2. ブラウザを更新しましたか？（F5）
3. 色コードは `#` から始まっていますか？
4. クォーテーション `"` で囲まれていますか？

### エラーが出る

1. カンマ `,` を忘れていませんか？
2. 括弧 `{ }` や `[ ]` の数は合っていますか？
3. 最後のテーマにカンマをつけていませんか？

### 開発者ツールで確認

1. ブラウザで F12 キーを押す
2. 「Console」タブを開く
3. 赤いエラーメッセージがないか確認

## ふろく：配列の便利な使い方

### 配列の長さを取得

```javascript
const colors = ["赤", "青", "緑"];
console.log(colors.length);  // 3
```

### 配列の最後に追加

```javascript
const colors = ["赤", "青"];
colors.push("緑");
console.log(colors);  // ["赤", "青", "緑"]
```

### 配列をループで回す

```javascript
const colors = ["赤", "青", "緑"];

for (let i = 0; i < colors.length; i++) {
    console.log(colors[i]);
}
// "赤" "青" "緑" が順番に表示される
```
