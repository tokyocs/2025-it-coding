# サンプルファイル - 自己紹介サイト

このフォルダには、第1回の授業で作る自己紹介サイトのサンプルファイルが入っています。

## 📁 ファイル構成

```
example/
├── index.html    ← ページの構造（HTML）
├── style.css     ← デザイン（CSS）
└── script.js     ← 動作のプログラム（JavaScript）
```

---

## 🎯 このサンプルで学べること

### HTML

- ページの基本構造
- 見出し（`<h1>`, `<h2>`）の使い方
- 段落（`<p>`）の使い方
- リスト（`<ul>`, `<li>`）の使い方

### CSS

- 背景色の設定
- 文字の色と大きさ
- ボックスのスタイリング
- 余白（padding, margin）の使い方

### JavaScript

- ボタンのクリックイベント
- アラートの表示
- console.logの使い方

## 💡 使い方

### 🌟 初めての人向け：ファイルの開き方

**ステップ1：このフォルダを見つける**
- ダウンロードしたフォルダの中の `20251020` → `example` フォルダを開いてください
- 3つのファイル（index.html, style.css, script.js）が見えるはずです

**ステップ2：HTMLファイルをブラウザで開く**

方法は2つあります：

**方法A：ダブルクリック**
1. `index.html` をダブルクリック
2. 自動的にブラウザ（Chrome、Safari、Edgeなど）が開きます

**方法B：右クリックから開く**
1. `index.html` を右クリック
2. 「プログラムから開く」を選ぶ
3. ブラウザ（Chrome、Safari、Edgeなど）を選ぶ

これで自己紹介サイトが表示されます！🎉

---

### ステップ1：サンプルを開く

1. 上の手順で `index.html` をブラウザで開く
2. ボタンをクリックしてみる
3. どんな動きをするか確認する

### ステップ2：コードを読む

**コードを見るには：**

**方法A：メモ帳で開く（簡単）**
1. ファイル（例：index.html）を右クリック
2. 「プログラムから開く」→「メモ帳」を選ぶ
3. コードが見えます！

**方法B：Visual Studio Codeで開く（推奨）**
1. Visual Studio Codeを開く
2. 「ファイル」→「フォルダーを開く」
3. `example` フォルダを選ぶ
4. 左側に3つのファイルが表示されます

どちらでもOK！まずはメモ帳で見てみるのもいいよ。

### ステップ3：カスタマイズする

ファイルを編集して、自分だけのサイトを作ってみよう！

**編集の仕方：**
1. ファイルをメモ帳かVisual Studio Codeで開く
2. 文字を変更する
3. ファイルを保存する（Ctrl+S または Cmd+S）
4. ブラウザで更新する（F5キー）

**変更してみよう：**

**HTML（index.html）:**
- 名前を自分の名前に変える
- 趣味を自分の趣味に変える
- 好きなものを追加する

**CSS（style.css）:**
- 背景色を変える（`background-color`）
- 文字の色を変える（`color`）
- ボタンの色を変える

**JavaScript（script.js）:**
- アラートのメッセージを変える
- console.logのメッセージを追加する

## 困ったときは

1. **ファイルが開けない**
   - ファイルの場所を確認しましょう
   - Visual Studio Codeでフォルダごと開いてみましょう

2. **変更が反映されない**
   - ファイルを保存しましたか？（Ctrl+S または Cmd+S）
   - ブラウザを更新しましたか？（F5キー）

3. **ボタンが動かない**
   - 開発者ツール（F12）でエラーがないか確認
   - script.jsが正しく読み込まれているか確認
