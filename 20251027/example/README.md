## 💡 使い方

### 🌟 初めての人向け：ファイルの開き方

**ステップ1：このフォルダを見つける**
- ダウンロードしたフォルダの中の `20251027` → `example` フォルダを開いてください
- 3つのファイル（index.html, style.css, script.js）が見えるはずです

**ステップ2：HTMLファイルをブラウザで開く**

**方法A：ダブルクリック（簡単！）**
1. `index.html` をダブルクリック
2. 自動的にブラウザが開きます

**方法B：右クリックから開く**
1. `index.html` を右クリック
2. 「プログラムから開く」を選ぶ
3. ブラウザ（Chrome、Safari、Edgeなど）を選ぶ

これでカラフルなボックスが表示されます！🎉

---

### ステップ1：サンプルを見る

1. 上の手順で `index.html` をブラウザで開く
2. 3つのカラフルなボックスを確認する
3. それぞれの色や文字の大きさを見る

### ステップ2：CSSを変更してみる

**CSSファイルを開く：**
1. `style.css` を右クリック
2. 「プログラムから開く」→「メモ帳」または「Visual Studio Code」を選ぶ

**以下を試してみましょう：**

**練習1：背景色を変える**

```css
.box1 {
    background-color: #ffd700; /* 金色に変えてみよう！ */
}
```

**練習2：文字色を変える**

```css
.box2 {
    color: #ff0000; /* 赤色に変えてみよう！ */
}
```

**練習3：文字の大きさを変える**

```css
h1 {
    font-size: 48px; /* もっと大きくしてみよう！ */
}
```

### ステップ3：自分だけのボックスを作る

HTMLに新しいボックスを追加して、CSSでスタイルをつけてみよう！

**手順：**
1. `index.html` をメモ帳などで開く
2. 下のコードをコピーして、`</body>` の前に貼り付ける
3. 保存する（Ctrl+S または Cmd+S）
4. 次に `style.css` を開いて、CSSも追加する
5. ブラウザで更新する（F5キー）

**HTML（index.html）に追加：**

```html
<div class="my-box">
    <h2>私のボックス</h2>
    <p>これは私が作ったボックスです！</p>
</div>
```

**CSS（style.css）に追加：**

```css
.my-box {
    background-color: #あなたの好きな色;
    color: #あなたの好きな文字色;
    padding: 20px;
    margin: 20px 0;
    border-radius: 10px;
}
```

## 困ったときは

1. **CSSが効かない**
   - ファイル名が `style.css` になっているか確認
   - HTMLに `<link rel="stylesheet" href="style.css">` があるか確認
   - ファイルを保存してブラウザを更新

2. **色が変わらない**
   - カラーコードが正しいか確認（`#` を忘れずに！）
   - セミコロン `;` を忘れていないか確認
   - クラス名が正しいか確認

3. **変更が反映されない**
   - ファイルを保存しましたか？（Ctrl+S または Cmd+S）
   - ブラウザを更新しましたか？（F5キー）
   - キャッシュをクリアしてみる（Ctrl+Shift+R または Cmd+Shift+R）
