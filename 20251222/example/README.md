# まとめクイズアプリ

## 📁 ファイル構成

```
example/
├── index.html        ← クイズページ（これを開く）
├── style.css         ← デザイン
├── script.js         ← クイズのプログラム
├── quiz-maker.html   ← クイズを作るページ
├── quiz-maker.css    ← クイズメーカーのデザイン
├── quiz-maker.js     ← クイズメーカーのプログラム
└── README.md         ← このファイル
```

## 🚀 使い方

### 1. クイズを開始する

1. `index.html` をダブルクリックしてブラウザで開く
2. クイズが自動的に始まります！

### 2. 問題に答える

**選択式問題：**

1. 正しいと思う答えをクリック
2. 「答えを確認」ボタンを押す
3. 正解は緑色、不正解は赤色で表示

**穴埋め問題：**

1. 空欄にコードを入力
2. 「答えを確認」ボタンを押す（または Enter キー）
3. 正解なら緑、不正解なら正解が表示される

### 3. カテゴリを選ぶ

上部のボタンで、特定のカテゴリだけに絞って練習できます：

- **全部** - 全15問
- **HTML** - 3問
- **CSS** - 3問
- **JavaScript** - 3問
- **ファイル構成** - 3問
- **配列** - 3問

### 4. もう一度挑戦

結果画面の「🔄 もう一度挑戦する」ボタンで最初から始められます。
問題の順番はランダムに変わります！

## 📚 クイズの内容

### HTML (3問)

- 見出しタグ（h1）
- ボタンタグ（button）
- CSSの読み込み（link）

### CSS (3問)

- 背景色（background-color）
- 文字色（color）
- 文字サイズ（font-size）

### JavaScript (3問)

- 変数（let）
- 要素の取得（getElementById）
- イベント（addEventListener）

### ファイル構成 (3問)

- HTML/CSS/JSの役割
- scriptタグの位置
- フォルダパス

### 配列 (3問)

- 配列のインデックス
- length プロパティ
- Math.random()

## 🛠️ クイズメーカー

### 使い方

1. `quiz-maker.html` を開く
2. 「作成する」タブで問題を作成
3. 「プレビュー」タブで動作確認
4. 「マイクイズ」タブで一覧表示・コード出力

### 機能

- **選択式問題** - 4つの選択肢から正解を選ぶ形式
- **穴埋め問題** - コードの一部を入力する形式
- **プレビュー** - 作った問題を実際に解いてみる
- **コード出力** - JavaScriptのコードとして出力
- **ローカル保存** - 作った問題はブラウザに保存される

---

## 🔧 カスタマイズしてみよう

`script.js` の `allQuestions` 配列に、自分で問題を追加することもできます！

```javascript
{
    id: 16,
    category: "html",
    categoryLabel: "HTML",
    type: "choice",
    question: "ここに問題文を書く",
    options: ["選択肢A", "選択肢B", "選択肢C", "選択肢D"],
    correct: 0,  // 正解の番号（0から始まる）
    explanation: "解説を書く"
}
```
