# サンプルプロジェクト

このフォルダには、3つの異なるファイル構成のサンプルが入っています。
それぞれのサンプルを開いて、ファイル構成の違いを比べてみよう！

---

## 📁 サンプルの種類

### 1. basic/ - 基本的な構成

**フォルダ構成：**
```
basic/
├── index.html
├── style.css
└── script.js
```

**特徴：**
- すべてのファイルが同じフォルダにある
- 小規模なプロジェクトに最適
- ファイルの読み込みが簡単

**こんなときに使おう：**
- 1ページだけのウェブサイト
- 練習用のプロジェクト
- 簡単なアプリ

**開き方：**
1. `basic` フォルダを開く
2. `index.html` をブラウザで開く

---

### 2. organized/ - 整理された構成

**フォルダ構成：**
```
organized/
├── index.html
├── css/
│   └── style.css
├── js/
│   └── script.js
└── images/
    └── （画像ファイル）
```

**特徴：**
- ファイルの種類ごとにフォルダで整理
- 中規模なプロジェクトに最適
- どこに何があるか分かりやすい

**こんなときに使おう：**
- 3-10ページくらいのウェブサイト
- 画像やファイルが増えてきたとき
- チームで作業するとき

**開き方：**
1. `organized` フォルダを開く
2. `index.html` をブラウザで開く

**ポイント：**
- パスは `css/style.css` のようにフォルダ名も含める

---

### 3. advanced/ - 発展的な構成

**フォルダ構成：**
```
advanced/
├── index.html
├── about.html
└── assets/
    ├── css/
    │   ├── common.css
    │   ├── home.css
    │   └── about.css
    ├── js/
    │   ├── common.js
    │   └── home.js
    └── images/
        └── （画像ファイル）
```

**特徴：**
- 複数のページを持つ
- 共通のスタイルとページごとのスタイルを分けている
- すべての資材を `assets` フォルダにまとめている
- 大規模なプロジェクトに最適

**こんなときに使おう：**
- 10ページ以上のウェブサイト
- 複雑な機能を持つアプリ
- 本格的なプロジェクト

**開き方：**
1. `advanced` フォルダを開く
2. `index.html` をブラウザで開く
3. ナビゲーションバーから「説明」ページにも行ってみよう

**ポイント：**
- パスは `assets/css/common.css` のように深くなる
- 共通のスタイルを複数のページで再利用できる

## 🎯 学習の進め方

### ステップ1：基本的な構成を見る

1. `basic/index.html` をブラウザで開く
2. `basic` フォルダの中のファイルを見てみる
3. HTMLからCSSとJavaScriptがどう読み込まれているか確認

### ステップ3：発展的な構成を見る

1. `advanced/index.html` をブラウザで開く

1. 好きなサンプルを選ぶ
2. テキストや色を変えてみる
3. 新しい要素を追加してみる
4. ブラウザで結果を確認

## 🔍 よくある質問

### Q1: どの構成から始めればいい？

**A:** 最初は **basic** から始めましょう！
慣れてきたら、**organized** や **advanced** に挑戦してみてください。

### Q2: フォルダの名前は変えてもいい？

**A:** はい、変えてもOKです！
ただし、HTMLの `href` や `src` も同じように変える必要があります。

例：
- `css/` を `styles/` に変えたら
- `<link href="styles/style.css">` に変える

### Q3: 自分のプロジェクトはどの構成にすればいい？

**A:** プロジェクトの大きさで決めましょう：
- **小さい**（1-3ページ）→ **basic** または **organized**
- **中くらい**（3-10ページ）→ **organized**
- **大きい**（10ページ以上）→ **advanced**
