# サンプルファイル

このフォルダには、第3回の授業で使うサンプルファイルが入っています。

## 📁 フォルダ構成

```
example/
├── counter/          ← カウンターアプリのサンプル
│   ├── index.html
│   ├── style.css
│   └── script.js
└── omikuji/          ← おみくじアプリのサンプル
    ├── index.html
    ├── style.css
    └── script.js
```

---

## 🎯 サンプルの種類

### 1. counter/ - カウンターアプリ

**内容：**

- ボタンを押すと数字が増える/減る
- リセットボタンで0に戻る

**ファイル：**

- `index.html` - ページの構造
- `style.css` - デザイン
- `script.js` - 動作のプログラム

**開き方：**

1. `example` フォルダの中の `counter` フォルダを開く
2. `index.html` をダブルクリック（または右クリック→ブラウザで開く）
3. カウンターアプリが表示されます！

**学べること：**

- イベントリスナーの使い方
- 変数の更新
- DOMの操作

---

### 2. omikuji/ - おみくじアプリ

**内容：**

- ボタンを押すとランダムにおみくじの結果が表示される
- 結果によって背景の色が変わる

**ファイル：**

- `index.html` - ページの構造
- `style.css` - デザイン
- `script.js` - 動作のプログラム

**開き方：**

1. `example` フォルダの中の `omikuji` フォルダを開く
2. `index.html` をダブルクリック（または右クリック→ブラウザで開く）
3. おみくじアプリが表示されます！

**学べること：**

- 配列の使い方
- ランダムな数の生成
- クラスの追加/変更

## 💡 使い方

### 🌟 初めての人向け：まず動かしてみよう！

**カウンターアプリを開く：**
1. `20251117` → `example` → `counter` フォルダを開く
2. `index.html` をダブルクリック
3. ブラウザで開いたら、ボタンを押してみよう！

**おみくじアプリを開く：**
1. `20251117` → `example` → `omikuji` フォルダを開く
2. `index.html` をダブルクリック
3. おみくじボタンを押してみよう！

---

### ステップ1：サンプルを見る

1. まずはブラウザで開いて動作を確認
2. どんな機能があるか試してみる
3. ボタンを何回か押して、どう動くか観察する

### ステップ2：コードを読む

**ファイルを開く方法：**

**方法A：メモ帳で開く（簡単）**
- ファイルを右クリック → 「プログラムから開く」 → 「メモ帳」

**方法B：Visual Studio Codeで開く（推奨）**
- Visual Studio Codeを起動 → 「ファイル」→「フォルダーを開く」→ `counter` または `omikuji` フォルダを選ぶ

**読む順番：**
1. まず `index.html` を読む（ページの構造）
2. 次に `style.css` を読む（見た目）
3. 最後に `script.js` を読む（動き）

### ステップ3：カスタマイズする

**編集の手順：**
1. ファイルをメモ帳などで開く
2. 変更したい部分を編集する
3. 保存する（Ctrl+S または Cmd+S）
4. ブラウザで更新する（F5キー）
5. 変更が反映されているか確認

**試してみよう：**
- 色を変えてみる
- テキストを変えてみる
- 数字を10ずつ増やすようにしてみる
