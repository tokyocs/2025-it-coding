## 📝 カウンターアプリのプロンプト

### 基本版（初心者向け）

```
### タスク ###
カウンターアプリを作成してください。

### 条件 ###
- HTML, CSS, JavaScriptのファイルを別々に作成してください
- 画面に数字を表示してください（最初は0）
- 「＋ ふやす」ボタンをクリックすると、数が1増えるようにしてください
- 「－ へらす」ボタンをクリックすると、数が1減るようにしてください
- 「リセット」ボタンをクリックすると、数が0に戻るようにしてください
- 見た目をきれいにしてください（色、フォント、レイアウトなど）
```

## 🎲 おみくじアプリのプロンプト

### 基本版（初心者向け）

```
### タスク ###
おみくじアプリを作成してください。

### 条件 ###
- HTML, CSS, JavaScriptのファイルを別々に作成してください
```

## 💡 プロンプトのコツ

### ✅ 良いプロンプトの例

1. **具体的に書く**
   - ❌ 「カウンターを作って」
   - ✅ 「+ボタンと-ボタンがあるカウンターアプリを作ってください」

2. **やりたいことを明確にする**
   - ❌ 「きれいにして」
   - ✅ 「背景を青色にして、ボタンを丸くしてください」

3. **ファイルを分けることを伝える**
   - ✅ 「HTML, CSS, JavaScriptのファイルを別々に作成してください」

### 🔧 うまくいかないときのプロンプト

**エラーが出たとき**
```
上のコードでエラーが出ました。
以下のエラーメッセージが表示されています：
[エラーメッセージをここに貼る]
修正してください。
```

**動きを変えたいとき**
```
今のコードで、[変えたい部分]を[どう変えたいか]にしてください。
例：今のコードで、ボタンの色を赤から青に変えてください。
```

**機能を追加したいとき**
```
今のコードに、[追加したい機能]を追加してください。
例：今のコードに、音が鳴る機能を追加してください。
```

## 📚 参考：よく使う言葉

| やりたいこと | プロンプトで使う言葉 |
|------------|-------------------|
| ボタンを押したら何かする | 「ボタンをクリックすると〜してください」 |
| ランダムに選ぶ | 「ランダムに表示してください」 |
| 色を変える | 「背景色を〜に変えてください」 |
| 数を増やす/減らす | 「数を1増やしてください」「数を1減らしてください」 |
| アニメーション | 「アニメーションをつけてください」「動きをつけてください」 |
| 条件によって変わる | 「〜のとき、〜してください」 |
