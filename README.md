## 🚀 初めての人へ - クイックスタート

このリポジトリをダウンロードしたら、まず何をすればいいか説明します！

### ステップ1：フォルダを開く

1. ダウンロードしたフォルダを開く
2. 日付の入ったフォルダ（`20251020`、`20251027` など）が見えます

### ステップ2：サンプルを動かしてみる

**第1回から始める場合：**
1. `20251020` フォルダを開く
2. `example` フォルダを開く
3. `index.html` をダブルクリック
4. ブラウザで自己紹介サイトが開きます！🎉

**他の回を試したい場合：**
- 各フォルダの `example` フォルダの中に、すぐ動かせるサンプルが入っています
- `index.html` をダブルクリックすれば開きます

### ステップ3：詳しく学ぶ

各フォルダの `README.md` を開くと、詳しい説明があります：
- 20251020/README.md - HTML、CSS、JavaScriptの基礎
- 20251027/README.md - CSSの使い方
- 20251117/README.md - JavaScriptでページを動かす方法
- 20251201/README.md - ファイル構成の理解

**README.mdの開き方：**
- ダブルクリックで開く（ブラウザまたはメモ帳で表示されます）
- Visual Studio Codeで開くとさらに読みやすいです

## このリポジトリについて

東京コミュニティスクール初等部、G5のITの時間でコーディングを学ぶためのリポジトリです。
各回の授業内容とドキュメントは、日付ごとのディレクトリに整理されています。

**各フォルダの構成：**
```
20251020/（日付フォルダ）
├── README.md          ← 授業の説明
└── example/           ← すぐ動かせるサンプル
    ├── README.md      ← サンプルの使い方
    ├── index.html
    ├── style.css
    └── script.js
```

## 💡 困ったときは

### ファイルが開けない
- ファイルをダブルクリックしてみてください
- 右クリック → 「プログラムから開く」でブラウザを選んでください

### コードを編集したい
- ファイルを右クリック → 「プログラムから開く」→ 「メモ帳」を選ぶ
- Visual Studio Codeがあればもっと便利です

### 変更が反映されない
- ファイルを保存しましたか？（Ctrl+S または Cmd+S）
- ブラウザを更新しましたか？（F5キー）

## 🎯 学習の進め方

1. **まずサンプルを動かす** - `example` フォルダのファイルをブラウザで開く

3. **変更してみる** - 色やテキストを変えてみる
4. **自分で作る** - サンプルを参考に、自分のページを作る
